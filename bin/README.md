# bin/

This folder contains pre-compiled static Linux x86_64 binaries bundled for Render deployment.

| File    | Source |
|---------|--------|
| `fpcalc` | [Chromaprint v1.5.1 Linux x86_64](https://github.com/acoustid/chromaprint/releases/tag/v1.5.1) |

`ffmpeg` is provided via the `ffmpeg-static` npm package (no binary needed here).
